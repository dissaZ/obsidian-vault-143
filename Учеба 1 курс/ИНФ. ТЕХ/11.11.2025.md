# Тексты и шрифты
все шрифты делятся на два типа
- с засечками
- без засечек

Моно жирный шрифт - rel

Text - Transform
управление регистром 
1) in hred
2) none
3) upper case - прописные - высший регистр
4) lower case - строчные - нижний регистр
5) capitalize - первый символ каждого слова будет с заглавной 
6) font variant - капитель - все буквы отображаются как заглавные, но размером как прописные
7) text decorations - добавляет оформление текста (~~мигание~~ подчерк, и всякие как в .md) применять можно до 1 стиля через пробел (типа блинк не юзаем пон)
8) latter spatting/world spatting - расстояние между символами или между словами ( можно и минусовые )
9) text shadow - (горизонтальное смещение, вертикальное смещение, размытость, цвет ) 
10) line height - межстрочный интервал
11) text allign - выравнивание ( слева справа по центру )
12) text indent - отступ в каждом новом параграфе
13) list - style - type - - изменение маркеров списка
14) style position - ну типа список внутри или не внутри хз
15) list style image - ссылка на картинки ( замена маркеров на картинке )
16) display - блочный элемент сделает строчным или наоборот
17) border radius - округляет углы
18) box shadoow - тень во внутрь элемента или внаружу крч загугли это треш какойто
19) weight & height - ширина и высота (отдельно)
20) box sizing - html book читай
==Никогда не задайте заранее ширину и высоту== 
21) overflow - управляет отображением содержанием если оно целиком не помещается и выходит за область заданного размера
22) float - выравнивание элемента ( устарело )
23) clear - сброс обтеканий
