[[TРПО]]

Источник данных - продукт, в который пользователь вводит данные.

Приемник данных - система, которая является получателем данных.

# Протоколы
Это правила передачи информации в сети.

| Http   | Передачи гипертекстовых сообщений |
| ------ | --------------------------------- |
| Ftp    | Передачи файлов                   |
| Smtp   | Протокол передачи почты           |
| Pop    | Получение почты                   |
| Telnet | Удаленного доступа                |
| Ip     | Передача данных                   |
| Tcp    | Передача Данных между пк          |
| Dns    | Определение уникального ip        |

# Отладка программных продуктов. Инструменты отладки. Отладочные классы.

**Отладка** - это процесс обнаружение причин возникновения ошибки и ее последующего исправления.

**Тестирование** - это процесс обнаружения самого факта существования ошибки
