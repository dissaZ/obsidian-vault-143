# Интерпретатор командной строки ОС MS Windows
#интерпретатор #команднаястрока #Windows

Интерпретатор командной строки Windows - это программа, которая принимает от пользователя текстовые команды, интерпретирует ( то есть, понимает, обрабатывает и выполняет их, а затем, выводит результат своего выполнения обратно в текстовом виде)

Оболочка командной строки windows характеризуется следующими особенностями:

- реализует директивную парадигму программирования
- выполняется в режиме построчной интерпретации
- содержит управляющие конструкции и несколько видов циклов ( включая специализированные для обработки текстовых файлов)
- имеет оператор присваивания значений переменных
- поддерживает использование внешних программ в качестве операторов с обработкой кодов возврата
- работает с нетипизированными переменными, которые декларируются при первом упоминании (значения могут интерпретироваться как числа и использоваться в целочисленной арифметике)


| Команда              | Описание                                                               | Пример                 |
| -------------------- | ---------------------------------------------------------------------- | ---------------------- |
| cd/chdir             | показать текущую директорию, где вы сейчас                             | cd                     |
| cd <путь>            | сменить текущую директорию на указанную                                | cd C:\Users...         |
| cd ..                | перейти на уровень вверх                                               | cd ..                  |
| cd \                 | перейти в корень текущего диска из C:\f,fkgdfk\sgdg до C:              | cd \                   |
| dir                  | Показать список файлов и папок в текущей директории                    | dir                    |
| dir /w               | показать список в широком формате                                      | dir /w                 |
| tree                 | показать дерево каталогов, начиная с текущей папки                     | tree                   |
| мкdir                | создать новую папку                                                    | мкdir new_folder       |
| rmdir <имя>          | удалить **пустую** папку                                               | rmdir Old_folder       |
| copy <откуда> <куда> | скопировать файл/ы                                                     | copy file.txt C:\kgkrg |
| move <откуда> <куда> | переместить файлы или переименовать                                    | move old.txt new.txt   |
| del <имя>            | удалить файлы                                                          | del file.txt           |
| type <имя>           | вывести содержимое текстового файла на экран                           | type readme.txt        |
| <команда> /?         | Справка выведет полный список всех доступных флагов для любой команды. | dir /?; copy /? и тд   |
