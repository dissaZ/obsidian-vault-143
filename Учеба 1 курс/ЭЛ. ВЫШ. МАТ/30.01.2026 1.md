2.  
$$\lim_{x\to 7} \left(\frac{14}{x^2 - 49} - \frac{1}{x - 7}\right)$$

 Шаг 1: Приведение к общему знаменателю $(x^2 - 49 = (x - 7)(x + 7))$
$$\lim_{x\to 7} \left(\frac{14}{(x - 7)(x + 7)} - \frac{1(x + 7)}{(x - 7)(x + 7)}\right)$$

$$\lim_{x\to 7} \frac{14 - x - 7}{(x - 7)(x + 7)} = \lim_{x\to 7} \frac{7 - x}{(x - 7)(x + 7)}$$

 Шаг 2: Сокращение $7 - x = -(x - 7)$
$$\lim_{x\to 7} \frac{-(x - 7)}{(x - 7)(x + 7)} = \lim_{x\to 7} \frac{-1}{x + 7}$$

 Шаг 3: Вычисление предела
$$\frac{-1}{7 + 7} = \mathbf{-\frac{1}{14}}$$
3.  
$$\lim_{x\to 7} \left(\frac{14}{x^2 - 49} - \frac{1}{x - 7}\right)$$

 Шаг 1: Приведение к общему знаменателю $(x^2 - 49 = (x - 7)(x + 7))$
$$\lim_{x\to 7} \left(\frac{14}{(x - 7)(x + 7)} - \frac{1(x + 7)}{(x - 7)(x + 7)}\right)$$

$$\lim_{x\to 7} \frac{14 - x - 7}{(x - 7)(x + 7)} = \lim_{x\to 7} \frac{7 - x}{(x - 7)(x + 7)}$$

 Шаг 2: Сокращение $7 - x = -(x - 7)$
$$\lim_{x\to 7} \frac{-(x - 7)}{(x - 7)(x + 7)} = \lim_{x\to 7} \frac{-1}{x + 7}$$

 Шаг 3: Вычисление предела
$$\frac{-1}{7 + 7} = \mathbf{-\frac{1}{14}}$$
4.  Исходное выражение:
$$\lim_{x\to \infty} \left(\sqrt{3x^2+7} - \sqrt{3x^2+2x}\right)$$

 Шаг 1: Умножение на сопряженное выражение
$$\lim_{x\to \infty} \frac{(\sqrt{3x^2+7} - \sqrt{3x^2+2x})(\sqrt{3x^2+7} + \sqrt{3x^2+2x})}{\sqrt{3x^2+7} + \sqrt{3x^2+2x}}$$

 Используем формулу разности квадратов и упрощаем числитель:
$$\lim_{x\to \infty} \frac{(3x^2+7) - (3x^2+2x)}{\sqrt{3x^2+7} + \sqrt{3x^2+2x}} = \lim_{x\to \infty} \frac{7 - 2x}{\sqrt{3x^2+7} + \sqrt{3x^2+2x}}$$

 Шаг 2: Деление числителя и знаменателя на старшую степень (x)
$$\lim_{x\to \infty} \frac{\frac{7}{x} - 2}{\sqrt{3+\frac{7}{x^2}} + \sqrt{3+\frac{2}{x}}}$$

 Шаг 3: Вычисление предела (члены(пиписьки) с $1/x$ и $1/x^2$ стремятся к нулю)
$$\frac{0 - 2}{\sqrt{3 + 0} + \sqrt{3 + 0}} = \frac{-2}{2\sqrt{3}} = -\frac{1}{\sqrt{3}}$$

 Рационализация знаменателя:
$$-\frac{1}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} = \mathbf{-\frac{\sqrt{3}}{3}}$$
