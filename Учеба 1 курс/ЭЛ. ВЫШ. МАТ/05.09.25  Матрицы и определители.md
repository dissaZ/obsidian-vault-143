# Матрицы и определители



<span style="background:#d3f8b6">матрица - это прямоугольная таблица образованная из элементов некоторого множества и состоящая из m строки n столбцов.</span>


Amxn=(a11 a12 a13... a1n / a21 a22 a23... a2n / am1 am2 am3... amn)
Amxn=(aij), i=1,m,  j-1,n,  aij-элем матр
$$
A_{m*n}=(a_{11} a_{12} a_{13})\dots a_{2n}
$$
$$
A_{m*n}=(a_{21} a_{22} a_{23})\dots a_{2n}
$$
$$
A_{m*n}=(a_{m_{1}} a_{m_{2}} a_{m_{3}})\dots a_{mn}
$$
$$
B_{m*n}=(b_{ij})
$$
Bmxn = (bij)

если m=n - матрица квадратная
число m=m - ее порядком

главной диагональю матрицы называется диагональ a11 a22 a33... ann
$$
a_{11} a_{22}{a_{33}}\dots a_{nn}
$$


побочных ( второстепенный ) диагональю матрицы называются -
a1n a2(n-1) ... an1 - второстеп. диаг.

матрицы A,B одного размера называются равными если они совпадают по элементно

# виды матриц

1) квадратная матрица у которой все элементы кроме элементов на главной диагонали равны 0 - называется диагональной 
2) диагональная матрица у которой все элементы главной диагонали равны 1, такая матрица называется единичной
I=E= ( 1 0 0 
       0 1 0  - 
       0 0 1)
3) матрица все элементы которой равны 0 называется нулевой
4) квадратная матрица называется верхне-треугольный если все элементы расположенные ниже главной диагонали равны нулю
A= ( 2 8 5
    0 4 6
    0 0 -1)
5) квадратная матрица называется нижне-треугольной если все элементы расположеные выше главной диагонали равны 0
A= (2 0 0 
	8 4 0
    5 6 -1)
6) матрица содержащая 1 столбец или одну строку называется вектором
	A= (a1
		a2
		am) - вектор столбец
