#циклы 

Цикл - это программная конструкция, позволяющая выполнять набор команд **несколько раз**, пока выполняется определённое условие.

Циклы можно классифицировать:

## по кол-ву повторений:

- с фиксированным числом итераций (арифметические)
- с неопределённым числом шагов (условные/итерационные)

## По положению условия в структуре:

- с проверкой перед выполнением (предусловие)
- с проверкой после (постусловие)
- с возможностью выхода из середины цикла

# Ключевые особенности:

- условие может быть любым логическим выражением
- проверка выполняется до начала очередной итерации
- если условие ложно изначально - тело не выполнится ни разу
- одинарные команды можно писать без фигурных скобок

#for
## for 

for - ( инициализация; условие; изменение значений переменных )

{
	действие;
}

## Особенности конструкции

- подходит для циклов с известным кол-вом повторов
- первая часть ( инициализация) выполняется один раз
- вторая часть ( условие ) проверяется при каждой итерации
- третья часть ( обновление) выполняется в конце каждой итерации
- все три части - необязательны, но точки с запятой должны быть

## Если условие отсутствует, либо результат условия ложный, значит цикл бесконечный.

# Do while

## Основные черты
#do #while
 
 - условие  проверяется после выполнения тела цикла
 - гарантированно выполняется хотя бы одна интерация
 - часто используется для проверки корректности пользовательского ввода
 - если тело цикла - одна инструкция, {} не нужны
