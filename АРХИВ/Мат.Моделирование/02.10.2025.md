# Правила
первый столбец таблицы содержит свободные члены разложение целевой функции и базисных переменных и даёт базисное решение

признаком допустимости решения является:
не отрицательность решения всех элементов первого столбца кроме индекса 0<sub>00</sub> 

первая строка таблицы содержит коэффициент разложения целевой функции не отрицательность всех элементов первой строки кроме элементов с индексом 00 является признаком оптимальности решения.

## Правила выбора нового базиса:
в базисные переводится свободная переменная x для которой коэффициент первой строки отрицателен 

если базисное решения является не допустимым, свободное переводится базисная переменная x для которой коэффициент первого столбца отрицательный.

Если базисное решения является допустимым в свободные переводится базисные переменная для которой отношение коэффициентов максимально.
