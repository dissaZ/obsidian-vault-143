[[ИНФ ТЕХ]]
Обязательный атрибут action - отвечает за то, куда будут отправлены данные после того как пользователь нажмет кнопку отправить занимается (backend'р)

Method - get, post 
Tr input - создает разные элементы интерфейса и обеспечить взаимодействие с пользователем

Name - имя поля, предназначено для того, чтобы обработчик формы мог идентифицировать поле.
>htmlbook

atribute value определяет значение элементов формы которая будет отправлена на сервер или получено с помощью клиентских скриптов

type - сообщает браузеру к какому типу относиться элемент формы
- password - будут вместо текста точки - при обновлении страницы сбрасывается
- reset - очистка полей форм - прописывать в тег button
- submit - отправить
- checkbox - галочка в квадрате
- radio - точка в кружочке

Label - устанавливает связь между определенной меткой и элементов форм
for - идентификатор элемента с которым имеет связь

textarea - элемент формы для сохранения области, но допускается перенос строк.
==htmlbook==
select - списки 
		Option
			name
	size - сколько пунктов видно сразу


multiple - разрешает выбирать несколько элементов списка
