# Метод Квайна-Мак-Класки 
суть методы 
- **Склеивание высших рангов** Полученные выражения с прочерками снова сравниваются между собой. Склеить можно только те элементы, у которых прочерки стоят в одних и тех же позициях. При этом они должны отличаться друг от друга ровно в одном числовом разряде. В новом члене на месте этого отличия тоже ставится прочерк.

- **Пометка использованных членов** Каждый раз, когда элемент участвует в склеивании, его нужно отметить галочкой или точкой. Те члены, которые ни разу не удалось склеить с соседями, называются **простыми импликантами**. Они являются кандидатами для включения в итоговую формулу.

- **Завершение этапа** Процесс повторяется до тех пор, пока не останется ни одной пары для склеивания. Обычно это происходит, когда в группах остается по одному элементу или когда позиции прочерков во всех выживших членах не совпадают.

# Правила объединение 
1) все единицы должны быть заключены в прямоугольные контуры
2) во всех клетках контура должны быть только единицы
3) число клеток в контуре должно быть кратное степени числа 2
4) контуры могут накладываться друг на друга
5) контуры всех клеток которых уже вошли другие контуры являются лишними
6) для получения наиболее простой формулы надо выбирать контуры с максимально возможным числом клеток 
7) каждому контуру соответствуют конъюнкции составленные из переменных значения которых не изменяются во всех клетках контура
