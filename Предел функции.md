# Предел функции
Пусть функция $$
y=f(x)
$$
определена в некоторой окрестности точки x_0, кроме, возможно, самой точки x_0 Число А называется пределом функции в точки x_0 ( или при x которая стремится к x_0 если сама функция f от x не определена в точки x_0).
	Если для любого положительного эпсилон найдется такое положительное число дельта что для всех x не равных x_0 удовлетворяющих условий неравенства модуль x - x_0 выполняется другое неравенство
